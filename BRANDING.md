# Branding and Attribution Policy

This file describes the intended product norm for hosted deployments of Daily
Chaos Chess.

## Repository linkback

If you host a public instance of this project, please include a visible link to:

- the original repository
- the original project name, if unchanged

Suggested placement:

- footer
- about page
- source link in the site chrome

## Why this exists

The code license governs code reuse. This document expresses the project's
branding and attribution expectation for public hosts so the original project
continues to receive credit and discoverability.

## Important note

This branding policy is not a substitute for legal advice and is not, by itself,
the same thing as an enforceable open-source license condition.
