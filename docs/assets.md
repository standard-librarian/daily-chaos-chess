# Generic Chaos Asset Sources

Good places to source generic, reusable 3D props for the board atmosphere:

- [Poly Pizza](https://poly.pizza/)
  - Large CC0 library and easy to search for portals, crystals, ruins, candles, skulls, books, and ritual props.
- [Kenney](https://kenney.nl/assets)
  - Great for clean stylized game-ready props and environment kits.
- [Quaternius](https://quaternius.com/)
  - Strong source for cohesive low-poly environment packs, ruins, and fantasy props.

## Suggested visual categories

- portals
- sigils
- crystals
- obelisks
- ruined columns
- candles and braziers
- ravens, skulls, masks

Keep these assets decorative. The canonical game state should remain driven by
structured world entities and action scripts, not by imported meshes.
