# Draft: AnarchyChess Launch Post

## Title

I made Daily Chaos Chess: a website where the top comment each day becomes canon and mutates the board

## Body

I’ve been building a weird little open-source project called **Daily Chaos Chess**.

The idea is simple:

- there’s one live chess board
- people submit prompts for what should happen next
- the top prompt each day wins
- that prompt becomes canon and permanently mutates the board

So it’s not normal chess. It’s more like community-directed board corruption with a memory.

Current stack:

- Next.js + Vercel
- React Three Fiber / Three.js
- Turso
- BDD-first tests with Cucumber + Vitest

Repo:

- https://github.com/standard-librarian/daily-chaos-chess

I’d love help from people who want to contribute weird visual ideas, replay/history polish, board gimmicks, or just make the site feel more chaotic and alive.

There are already `good first issue` tasks open if you want an easy entry point.

If this sounds like your kind of nonsense, I’d really appreciate a star so more contributors can find it.

Also very open to ideas for:

- absurd but readable board mutations
- better lore presentation
- fun visual chaos props
- ways to make the daily reveal hit harder
